# 待注销用户｜WLAN-0714

一个完整、可通关的中文网页 ARG。

温岚已经死亡，但她的账号仍在每天凌晨 03:17 登录。玩家接管一台隔离终端，在邮件、聊天、云盘、档案和系统日志之间调查这个不可能继续在线的用户，并决定最后如何处置她留下的身份。

> 本作品完全虚构，不对应任何真实人物、机构或事件。当前公网试玩已按创作者要求撤下，本仓库保持私有。

## 游戏内容

- 约 20–35 分钟单人流程
- 模拟桌面与七个叙事应用
- 邮件阅读、密码解锁、关键词检索与隐藏联系人
- 8 项证据及上下文提示
- 证据板与调查进度
- 3 个结局，可重置重玩
- 本地自动存档
- 桌面端与移动端响应式布局

## 本地运行

需要 Node.js `>=22.13.0`。

```bash
npm install
npm run dev
```

构建与检查：

```bash
npm run lint
npm test
```

## 项目结构

- `app/ArgExperience.tsx`：剧情、证据、谜题状态和三个结局
- `app/globals.css`：完整视觉系统与响应式布局
- `app/page.tsx`：作品入口
- `tests/rendered-html.test.mjs`：服务端渲染测试

## 技术栈

- React 19
- TypeScript
- vinext / Vite
- 原生 CSS
- localStorage 本地存档

## 调查流程

玩家需要在不同应用之间交叉取证：

1. 从死亡回执中找到云盘恢复口令。
2. 从照片中取得可检索地址。
3. 用门禁记录恢复隐藏联系人。
4. 对照不同书写习惯、设备签名和角色证词。
5. 确认自己为何被邀请进入终端。
6. 在注销、公开和接收交接之间作出选择。

仓库不在 README 中公开完整答案，具体逻辑可在源代码中审阅。
